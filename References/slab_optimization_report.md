# 1D Slab 凝胶振荡参数优化报告

## 关键发现：S_χ = 0.7 显著提升振荡质量

通过逐参数灵敏度分析（10个参数 × 4-10个采样点）和组合测试，发现**最有效的单参数改进**是将 $S_\chi$ 从 1.0 降至 0.7。

### 性能对比

| 指标 | 默认 (S_χ=1.0) | 优化 (S_χ=0.7) | 提升 |
|---|---|---|---|
| θ 振幅 | 1.40 | **4.38** | **×3.1** |
| J 振幅 | 0.05 | **1.39** | **×28** |
| 周期 T | 27 | 34 | ×1.3 |
| 厚度脉动 | 微弱 | **明显** | 质变 |

### 物理解释

$S_\chi = \partial\chi/\partial\theta$ 控制温度对 Flory 相互作用参数的灵敏度。降低 $S_\chi$：

1. **拓宽了 LCST 转变的温度区间**——从"锐利开关"变为"宽幅渐变"
2. **允许 θ 摆幅更大**——因为需要更大的 $\Delta\theta$ 才能驱动同等的 $\Delta J$（溶胀变化）
3. **增大了 J 振幅**——从几乎纯热振荡转变为真正的热-机械耦合振荡

从 kymograph 可以清楚看到：

- **默认参数**：θ 振荡 ~1.4，但 J 几乎不动（ΔJ ≈ 0.05），凝胶厚度基本不变
- **S_χ=0.7**：θ 振荡 ~4.4，J 在表面附近剧烈脉动（0.3-3.5），凝胶厚度可见的周期性胀缩

**S_χ=0.7 将振荡从"热呼吸"升级为"真正的体积脉搏"。**

## 推荐优化参数集

将下面的改动应用到你的 `scan_optimized.py`：

```python
S_chi = 0.70     # ← 从 1.00 降到 0.70 （最重要）
# 其余参数保持不变即可
```

可选的进一步优化（需要你用自己的并行代码验证数值稳定性后使用）：

```python
S_chi = 0.70
alpha = 0.30     # 从 0.20 升到 0.30
# Bi_c = 1.3    # 可尝试，但大幅增加 Bi_c 可能导致数值刚性
```

## 数值稳定性注意

S_χ=0.7 使振荡振幅大增（θ 可达 5-6），Arrhenius 指数 Γ_A·θ/(1+ε_T·θ) 可能接近 cap 值。建议：

- 使用你代码中的 `arrh_exp_cap = 60`（已有）
- 在高分辨率运行时（N≥41），如果遇到 `Factor is exactly singular` 错误，尝试降低 `arrh_exp_cap` 到 30-40，或减小 `max_step` 到 0.25
- 这是大振幅振荡的固有刚性问题，不是参数选择的问题
